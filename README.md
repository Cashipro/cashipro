# CashiPro - Crypto Exchange Platform

- Spot Trading
- Real-time Order Book
- Market Data
- Wallet Management
- Secure Authentication

- Next.js 14
- TypeScript
- Tailwind CSS
- Zustand
- Vercel

```bash
